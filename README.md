# 🧮 Sarvesh's Calculator

A modern and responsive calculator built using **HTML, CSS, and JavaScript** with support for **dark/light themes**, basic arithmetic operations, and extra mathematical functions.

---

## 📌 Features

✔ Basic arithmetic operations

- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)

✔ Extra mathematical functions

- Square (x²)
- Square Root (√)
- Percentage (%)
- Pi value (π)

✔ Utility functions

- AC (Clear all)
- DEL (Delete last digit)

✔ User Experience Features

- Dark Mode 🌙
- Light Mode ☀️
- Theme toggle button
- Responsive design
- Smooth UI
- Modern fonts (Poppins)

# 💻 How To Use

1. Open calculator

2. Enter numbers

3. Select operator

4. Press:

```text
=
```

to calculate result

5. Use:

```text
AC
```

to clear

6. Use:

```text
DEL
```

to delete one character

7. Toggle dark/light theme using top icon
